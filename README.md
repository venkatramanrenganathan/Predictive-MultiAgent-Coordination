# PREDICTIVE MULTI-AGENT COORDINATION
 A novel anticipation based multi-agent coordination protocol is simulated here.

 **Associated Paper:** Venkatraman Renganathan, Sabyasachi Mondal, Saurabh Upadhyay, `Predictive Consensus for Trustworthy Networked Systems`, Submitted to the Special Issue on `Enhancing Trustworthiness of Cyber-Physical Systems` in IEEE Letters to Control Systems Society.

## Dependencies
- Matlab

## Steps to Run the Code 
1. Run the `main_simulation.m` file, it will automatically load all the necessary data and will run the simulation
